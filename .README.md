# SCAKS

SCAKS: **S**hanghai **CA**talysis **K**inetics **S**oftware

## Install

1. Via pip (Not uploaded to PyPI yet):

    ```shell
    pip install scaks
    ```

2. From source

    ```shell
    git clone https://github.com/PytLab/scaks.git

    cd scaks
    python setup.py install
    ```

## Test

```shell
python setup.py test
```

or 

```shell
python -m scaks.tests.test_all
```
